# simple_rename
영상 이름과 자막의 이름을 자동으로 일치시켜주는 프로그램입니다.

## Usage
	python 3.6+ 버전이 설치된 컴퓨터에서 simple_rename.py를 실행시킵니다.
	설정하실 영상, 자막 파일이 있는 위치를 붙여넣으면 끝!

## Warning
	영상 파일은 반드시 2편 이상 있어야 합니다.
	에피소드를 제외한 모든 영상의 파일명은 다 동일해야 합니다.
	
	ex )
	[HorribleSubs] Animation File - 01 [1080p]
	[HorribleSubs] Animation File - 02 [1080p]
	이건 되고
	
	[LeopardSubs] Animation File - 01 [1080p]
	[HorribleSubs] Animation File - 02 [1080p]
	이건 안됨.

	자막 파일에도 01, 02와 같이 에피소드를 분류할 수 있는 넘버링이 되어있어야 합니다.
